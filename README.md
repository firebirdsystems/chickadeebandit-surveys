# Surveys

A [Chickadee Bandit](http://chickadeebandit.com) app.

Multi-question surveys for household decisions, trip planning, and feedback — text, multiple choice, rating scale, and yes/no questions, with results revealed once the survey closes.

## Features

- Build surveys with text, multiple choice, rating (1-5), and yes/no questions
- Anonymous or attributed responses, creator's choice
- Results shown as charts once the survey closes
- Notifies the household when a survey is created or closed

## Install

In your hub, go to **Apps → Install from URL** and paste:

```
https://github.com/firebirdsystems/chickadeebandit-surveys/releases/latest/download/bundle.json
```
